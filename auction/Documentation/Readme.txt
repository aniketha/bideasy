README

1. Download apache tomcat 6(https://tomcat.apache.org/download-60.cgi) and follow installation steps and install the software.

2. Download MySQL (http://dev.mysql.com/downloads/) and follow installation steps.


3. Open the documentation folder and open schema.doc to view the list of tables used in the project.

4. Create those tables and attributes in MySQL.

5.download mail.jar and activation.jar(http://www.oracle.com/technetwork/java/index-138643.html) and place them in lib folder of application

6. Start the apache tomcat server and type localhost:8080/auction and you will be able to access the whole website.
